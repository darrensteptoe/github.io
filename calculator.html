Volatility Inputs (Std Dev)

These control how much the engine randomizes each variable across simulations. Higher = more honest volatility, lower = more stable assumptions.
Tip: Start with DEFAULTS.md, sanity-check your setup using EXAMPLES.md. If results feel “too stable,” increase volatility or enable Movable Share.
